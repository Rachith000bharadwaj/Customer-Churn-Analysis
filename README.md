# Customer Churn Analysis Dashboard

## Overview

This project presents an interactive Customer Churn Analysis dashboard using Tableau and Flask.

## Features

- Customer Churn Rate
- Gender Analysis
- Contract Analysis
- Internet Service Analysis
- Payment Method Analysis
- Monthly Charges
- Tenure Analysis

## Technologies

- Tableau Public
- Flask
- HTML
- Python

## Run

```bash
pip install -r requirements.txt
python app.py
```

Open:

http://127.0.0.1:5000

## Deployment

Hosted on Render.
